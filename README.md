# SyCoaching — App (React + Vite + Firebase)

## Démarrer en local
```bash
npm install
npm run start
```
Puis ouvre http://localhost:5173

## Build
```bash
npm run build
npm run preview
```

## Netlify
- Branch: main
- Build command: npm run build
- Publish directory: dist
- Variable d'environnement incluse dans `netlify.toml`: VITE_FIREBASE_CONFIG
